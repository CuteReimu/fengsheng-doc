## 开始

本项目使用了 [VuePress Theme Hope](https://theme-hope.vuejs.press/zh/) 框架，需要 Node.js v18.19.0 以上版本。

推荐安装 Node.js v20。然后执行：

```sh
npm install
```

接下来就可以启动本地调试或者打包了。

### 本地调试

```sh
npm run docs:dev
```

### 打包

```sh
npm run docs:build
```
